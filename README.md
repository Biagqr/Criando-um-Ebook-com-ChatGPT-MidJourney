# Criando-um-Ebook-com-ChatGPT-MidJourney
Este repositório contém o projeto desenvolvido durante o laboratório “Criando um Ebook com ChatGPT &amp; MidJourney” da Digital Innovation One (DIO). O objetivo foi utilizar ferramentas de IA generativa para:  Criar os prompts  Gerar textos com o ChatGPT  Criar a capa com MidJourney  Montar o ebook final
